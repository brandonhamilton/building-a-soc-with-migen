Installing the tools

Before we begin, we need to setup the required tools and libraries

Python 3.3

Migen requires Python 3.3, and this guide assumes that this version is available on the path as python.

$ python -V
Python 3.3.1

Installing Migen

Migen is a python toolbox for building complex digital hardware.

git clone git@github.com:milkymist/migen.git
cd migen
python setup.py install

Installing Mibuild

Mibuild is a build system and board database for Migen-based FPGA designs

git clone git@github.com:milkymist/mibuild.git
cd mibuild
python setup.py install

Xilinx ISE command line tools

This guide assumes that the Xilinx ISE Design Tools have been installed. Mibuild will automatically source the Xilinx ISE environment setup script, and will call the various tools (xst, ngdbuild, map, par, bitgen) to synthesize, implement and generate the output FPGA bitstream. Ensure that the relevant ISE version that supports the targetted FPGA is available.
