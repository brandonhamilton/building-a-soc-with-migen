Building a System-on-Chip in python with Migen.

This is a simple guide documenting the process of creating a SoC in python using the migen toolbox. The SoC will be built for the Virtex-6 FPGA ML605 Evaluation Kit

The system will heavily follow the design of the milkymist-ng SoC

Requirements

  • python3
  • migen
  • mibuild
  • Xilinx ISE command line tools
