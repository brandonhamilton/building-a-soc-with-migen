Overview

This is a simple guide documenting the process of creating and implementian a System-on-Chip design in python using the Migen toolbox.

Migen is a python toolbox for building complex digital hardware.

  • The latest documentation and reference can be found online at http://migen.readthedocs.org/.
  • A first introduction to FPGA design using Migen is documented in a Tutorial
  • The Milkymist community is a very useful place to ask questions, get help and contribute to the project.

The SoC will be built to run on the Virtex-6 FPGA ML605 Evaluation Kit.

The system will heavily follow the design of the milkymist-ng SoC

Requirements

  • Python 3.3
  • Migen
  • Mibuild
  • Xilinx ISE command line tools
